# SmartTasker

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

## Overview

SmartTasker is an intelligent task management application designed to help you organize your daily tasks effortlessly. Leveraging AI, it prioritizes and automates reminders, making productivity seamless. Whether you're managing personal errands or team projects, SmartTasker adapts to your workflow.

## Features

- AI-powered task prioritization and scheduling
- Smart reminders based on deadlines and user habits
- Intuitive drag-and-drop interface
- Collaborative task sharing with team members
- Dark mode and customizable themes
- Offline mode with automatic sync

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **AI Integration:** OpenAI GPT-4 API
- **Authentication:** JWT

## Installation

1. Clone the repo:
   ```
   git clone https://github.com/jacobhug90/SmartTasker.git
   ```
2. Navigate into the directory:
   ```
   cd SmartTasker
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Setup environment variables in `.env` (API keys, DB connection strings)
5. Start the development server:
   ```
   npm run dev
   ```

## Usage

1. Open your browser and go to `http://localhost:3000`
2. Sign up or log in using your credentials
3. Create tasks, assign priorities, and set reminders
4. Use AI suggestions to optimize your schedule
5. Share tasks with collaborators and track progress in real-time
